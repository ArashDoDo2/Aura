# AuraMobileClient

یک کلاینت اندروید نمونه برای Aura با رابط کاربری مدرن و ساده.

## ویژگی‌ها
- ورود آدرس DNS Server و SessionID
- دکمه اتصال/قطع اتصال
- نمایش وضعیت اتصال
- پیام‌های راهنما و خطا
- طراحی بر پایه Material Design

## شروع سریع
این پروژه فقط UI اولیه است و قابلیت ارتباط واقعی DNS را ندارد (برای توسعه آماده است).

## ساختار پروژه
- app/src/main/java/com/example/auramobileclient/MainActivity.kt
- app/src/main/res/layout/activity_main.xml
- app/src/main/res/values/strings.xml
- ...

برای توسعه بیشتر، کافیست منطق ارتباط DNS را به MainActivity اضافه کنید.
